# Hallo Welt

"Hallo Welt" ist traditionell das erste Programm, das man in einer neuen Programmiersprache schreibt. Es gibt einfach einen Text auf dem Bildschirm aus.

## Der print() Befehl

`print()` ist eine eingebaute Funktion in Python, die Text oder Werte auf dem Bildschirm ausgibt.

```python
print("Hallo, Welt!")
```

**Ausgabe:**
```
Hallo, Welt!
```
